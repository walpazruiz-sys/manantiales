# Fuera Del Algoritmo

Fuera Del Algoritmo es un cuaderno de poesía y mirada interior.
Reúne textos, reflexiones y materiales sobre educación, escritura y experiencia humana.
Es un espacio pensado para leer con calma, explorar por secciones y volver cuando haga falta.

## Contenido

- Poesía y escritura.
- Educación y reflexión docente.
- Cuaderno de textos y apuntes.

## Tecnologías

- HTML.
- CSS.
- Netlify.

## Cómo verlo

Puedes abrir el archivo `index.html` directamente en el navegador.
Si el sitio está publicado, también puedes visitarlo desde la versión en línea.

## Estructura del proyecto

- `index.html`: página principal del sitio.
- `README.md`: descripción general del proyecto.
- Carpeta de recursos: imágenes, estilos o archivos adicionales, si los hubiera.

## Objetivo del sitio

Este sitio busca ofrecer una voz humana que escribe, piensa, duda, enseña y observa.
Su tono es literario, reflexivo y educativo.

## Mantenimiento

Este proyecto se actualiza de manera manual.
Cada cambio importante debe revisarse antes de publicarse para conservar la coherencia visual y textual.

## Autoría del espacio

Fuera Del Algoritmo funciona como identidad pública del sitio.
No se muestra un nombre personal en la portada.
